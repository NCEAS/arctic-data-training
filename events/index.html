Dataset Publishing with the Arctic Data Center (June 2018)

Dates: June 22, 2018 12:30 to 14:00 Location: Davos, Switzerland Venue: POLAR 2018 Open Science Conference This hands-on session will cover (1) Open data archives, especially the Arctic Data Center; (2) What science metadata is and how it can be used; (3) How data and code can be documented and published in open data archives; (4) Web-based submission, and (5) Submission using R (pending sufficient time). Course overview The Arctic Data Center conducts training in data science and management, both of which are critical skills for stewardship of data, software, and other products of research that are preserved at the Arctic Data Center.
